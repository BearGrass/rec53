# rec53top

`rec53top` 是 rec53 的本地终端手册。它直接读取一个 rec53 指标端点，把当前进程状态整理成固定的六面板运行视图。

这组文档的入口在这里：

- [页面与字段](pages.zh.md)
- [操作与导航](operations.zh.md)

## 覆盖内容

- 每个页面是什么意思
- 每个可见字段代表什么
- 怎么快速读当前状态
- TUI 不够时下一步该看哪里

## 边界

- 只看单个目标
- 只在本地终端使用
- 只读
- 不保留长期历史
- 不提供集群视图

更广的事故处理请看 [观测看板布局](../observability-dashboard.zh.md) 和 [运维排查清单](../operator-checklist.zh.md)。
