# rec53top

`rec53top` is the local terminal manual for rec53. It reads one rec53 metrics endpoint directly and turns the live process into a fixed six-panel operating view.

Use this directory as the entrypoint for the TUI docs:

- [Pages and fields](pages.md)
- [Operations and navigation](operations.md)

## What it covers

- what each page means
- what each visible field measures
- how to read the current state quickly
- where to go next when the TUI is not enough

## Boundaries

- single target only
- local terminal only
- read-only
- no long-term history store
- no fleet view

For broader incident work, see [Observability Dashboard](../observability-dashboard.md) and [Operator Checklist](../operator-checklist.md).
