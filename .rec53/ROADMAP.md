# rec53 路线图

## 版本规划

### v0.1.0 - 基础功能 (当前)

**核心功能**
- [x] 基本递归解析
- [x] UDP/TCP 支持
- [x] 缓存机制（含类型区分）
- [x] Prometheus 监控
- [x] 优雅关闭
- [x] CNAME 链追踪
- [x] Glue records 处理
- [x] IP 质量评估与预取

**代码质量**
- [x] 并发安全修复（atomic + RWMutex）
- [x] 测试覆盖率提升（目标 >60%）✅ 当前 ~60%

### v0.2.0 - 性能优化

**功能需求**
- [ ] 并发查询支持
- [ ] 查询流水线
- [ ] 缓存预取优化
- [ ] 负缓存支持
- [ ] EDNS(0) 完整支持

**非功能需求**
- QPS: 目标 50,000+ queries/sec
- 延迟: P99 < 50ms (缓存命中)
- 内存: < 500MB (10万缓存条目)

### v0.3.0 - 安全增强

- [ ] DNSSEC 验证
- [ ] DNS-over-TLS (DoT)
- [ ] 查询速率限制

### v0.4.0 - 高级特性

- [ ] DNS-over-HTTPS (DoH)
- [ ] 配置热加载
- [ ] 健康检查接口
- [ ] 源地址白名单
