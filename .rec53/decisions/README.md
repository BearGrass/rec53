# 架构决策记录 (ADR)

本目录记录项目中的重要架构决策。

## 什么是 ADR?

架构决策记录 (Architecture Decision Record) 是一种轻量级的文档形式，
用于记录项目中重要的架构决策及其背景。

## ADR 列表

暂无记录。

## ADR 模板

```markdown
# ADR-NNN: 决策标题

## 状态
[Proposed | Accepted | Deprecated | Superseded]

## 背景
描述导致此决策的情况和问题。

## 决策
描述所做的决策及其理由。

## 后果
描述此决策带来的影响（正面和负面）。

## 相关文件
- 相关代码文件列表
```
